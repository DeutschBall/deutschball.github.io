# 顺炮布局疑型总结




## 拆炮补相捉骑河车

??? info "典型局面"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260627163802808.png)

顺炮直车进三兵对横车局面下，黑棋正着走双正马然后双横车，如此形成顺炮直车两头蛇对双横车局面。

此时黑棋对挺3卒会被红骑河车针对，是另一个疑形。

此时黑棋肋车骑河抓兵也是疑形，会被红棋飞象利用。

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260627170044690.png)

此后红棋退八路炮平三打车从黑7路反击，或者飞相飞车。

为什么是退八路炮而不是退士角炮？其一是黑有车7平2兑炮缓解尴尬的手段，红退炮避免黑找台阶下。其二是红士角炮此后配合马三进四形成马后炮阵型。

## 车进士角抓炮

??? info "典型局面"
	
	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/202606041718903.png)

跳边马会导致炮无根，肋车可以进士角抓炮。

如图局面红黑反转一下，顺炮横车对直车时也可能遇到。

???+ info "边马离心"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260604224659507.png)

两者的区别在于，右图红方作为直车方多走了兵三进一一步棋，这会产生什么影响呢？

左图红棋有弃马十三着的骗着，而右图的黑方没有。

这是因为右图红方挺起三兵后，有车二进三到兵林抓炮的自救机会。

然而右图中挺起的三兵又暴露了新的问题：

黑车4进6抓炮，红炮八进二躲，黑车4退2骑河再抓炮，同时抓着三兵。黑骑河车杀三兵后还先手捉马。

## 骑河车抓探头炮

??? info "典型局面-红双正马探头炮"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260627171908541.png)

??? info "典型局面-红单提马探头炮"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260627172328256.png)


首先说红探头炮的目的。黑棋右马无法跳正，黑棋进3卒则红棋先平炮打车然后骑河车抓黑3卒。

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260627173331039.png)

借助巡河炮平七平九骚扰的先手，红棋想率先出动九路车。

然而上述两个局面红探头炮均没有稳定的立足之地，被黑骑河车拦截后会非常尴尬，面临被卒拱死的风险。

不管红炮八平七还是平九还是进兵交给马保，均无法善终。

目的一是腾开拐角马路线，此后红出九路车抓炮时黑正好拐角马保炮。二是让红炮没有兑换的目标，如果跳边马红炮走投无路可以换马，而飞边象黑炮顶多能换象。
