# 列炮布局疑型总结


## 贪冲卒大子慢被压马


??? info "典型局面"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614000735435.png)

此局面下黑棋仗着自己有担子炮，想不出车，先贪一个卒。

不管黑棋冲哪个卒，红车二进六压马黑棋就不好办。



### 局例-中炮对小列炮进7卒

1、炮二平五 炮2平5

2、马二进三 马8进7

3、车一平二 卒7进1？

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614000942756.png)

黑进7卒不禁起不到活通7路马的作用，并且会被红右车过河把7路马逼退后，红棋还能抢到兵七进一，如此黑棋两马都死。

红棋是一定能抢到兵七进一的，这一步制约黑3路马很关键。



4、车二进六 马2进3？

5、车二平三 马7退5

黑棋若走马3退5则炮八进五入侵布置线，黑棋崩溃

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614001316297.png)

此时红棋的后续手段是什么呢？

黑棋有窝心马，炮震五子是最具针对性的走法，直接走五八炮，黑棋可以冲3卒，因此红棋先挺七兵顺便制马，如此黑棋两马非常痛苦。

6、兵七进一 车1平2

7、马八进七 车2进6

8、车九进二！

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614001457506.png)

红高车保马，黑过河车就不足为惧了。

8、车二平三

9、炮八进四 卒3进1

10、炮八平七

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614001703658.png)

明显红大优。

如果黑棋在第4回合改走高车保马呢？


4、 ... 车9进2

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614002043299.png)

此时红棋又该如何考虑呢？

黑棋2路马只能跳正，如果跳边红棋有炮八进五入侵布置线得子。

因此红棋走兵七进一，黑棋明知道跳正马被制也得这样走。

红棋不怕黑马7进6再冲7卒的反击，黑中卒无根，红中炮打出后车二平三叫铁门栓有先手。

5、兵七进一 马2进3

6、马八进七 车1平2

7、车九平八 车2进6

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614002440970.png)

红棋先马七进六盘河了，接下来就要兵七进一反击，黑7路马也上不来，红优。

### 局例-中炮对小列手炮进3卒

1、炮二平五 炮2平5

2、马二进三 马8进7

3、车一平二 卒3进1？

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614002900297.png)

红棋直接骑河车抓卒，黑纯亏。

这不是压马的事情了，骑河车简明占优。



### 局例-红棋不出车勾引黑棋犯错

1、炮二平五 炮2平5

2、马二进三 马8进7

3、兵七进一

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614005559313.png)

红棋不出车，先挺七兵勾引黑对挺7卒，黑棋挺卒后红再车一平二，黑方就相当于落入贪了一步卒的陷阱，红棋就相当于抢到了兵七进一这关键一步。

黑棋只能走车9平8，如此红棋再出车黑棋就可以进炮封车了。

黑棋出车红棋可以抢两头蛇，或者马八进七出左车也来个前炮后车封车。


## 士角炮打串

??? info "典型局面"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614003417813.png)

小列炮红棋出车后，黑棋不出车跳双正马，此时红棋五六炮要求打串，黑棋不管是出车保炮，还是补士预防，都会按下葫芦浮起瓢引发其他问题。

### 局例-黑补士预防

1、炮二平五 炮2平5

2、马二进三 马8进7

3、车一平二 马2进3

4、炮八平六 士4进5

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614003752050.png)

黑棋补士后暴露的新问题，黑7路马无根。

如果此时黑棋走的是顺炮，那么7路马是有2路炮生根的，但是黑棋列炮，8路炮使不上劲。

红棋右车过河抓马即可占优

5、车二进六

黑棋有高车保马和拆中炮调整两种思路。

其中高车保马是错着。

??? failure "高车保马"

	5、... 车9进2

	6、炮六进二 卒9进1

	红巡河炮要求打死车，黑棋只能挺卒，但是红炮还能打马

	7、炮六平三

	??? info "局面图"

		![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614004237240.png)
	
	红棋得子。

	如果黑棋补的是6路士，此时黑马也可以被逼退，红炮至少能打个象，红棋也是大优。

??? question "拆炮调整"

	5、... 炮5平4

	??? info "局面图"

		![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614004407325.png)
	
	这种情况红棋再走车二平三无意义了，黑飞象给马生根接着就要退炮打车还有反击。

	此时红棋只需要以逸待劳，兵七进一制住另一匹马即可，黑7路马肩负着中卒的使命，也不敢随便跳。黑棋拆炮已经浪费一步棋让红棋占优了。


### 局例-黑出车保炮


1、炮二平五 炮2平5

2、马二进三 马8进7

3、车一平二 马2进3

4、炮八平六 车9平8

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614004852282.png)

此时红走士角炮串打，黑棋只能是打将抽回，但是要被兑窝车，并且红棋中路有攻势，也是红优。

5、炮六进五 炮5进4

6、马三进五 炮8平4

7、车二进九 马7退8

??? info "局面图"

	![](https://raw.githubusercontent.com/DeutschBall/picgo/main/20260614005053780.png)

之前很多对手走到这里，都会走马五进六抽马，但是黑士4进5后，可以出车抓死马，红棋优势荡然无存。

此时红棋就正常走马八进七出车就可以了。
