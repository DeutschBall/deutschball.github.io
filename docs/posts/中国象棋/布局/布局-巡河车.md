## 什么时候才能装逼？

什么局面下才可能存在这种下法

摆一下意思意思

首先说，为什么要先送兵再出车取回来？如果红先出巡河车，则黑立刻出直车邀兑窝车。红如果兑掉那么这个兵卒就兑不开了，红如果不兑那么躲车丢一先。

因此走个险着先送兵，这样右车巡河后，就不怕黑出直车邀兑了

??? note "局面图"
    ![image-20250810185056108](https://raw.githubusercontent.com/DeutschBall/picbed/main/image-20250810185056108.png)

然后什么情况下存在这种战术呢？

1. 送的是“远车兵”，比如送七兵就出二路车，送三兵就出八路车。
2. 对面先挺的卒，如图黑方3卒已经挺起，此时红方可以挺七兵送掉。
3. 送掉的兵后面不能有己方其他子力比如马，否则黑卒继续下冲就顶到了🥵🥵🥵
4. 黑卒后面不能有其他兵力支援，比如后面跟个大车，那就是铁棍子直接狠狠捅进来了💀💀💀
5. **送掉之后，黑方有没有手段保护这个黑卒，或者利用黑卒挣先手？这个必须具体场景具体分析了。**

## 为什么会装逼失败？

显然是这五条里面有不满足的条件，通常是条件5，因为我鼠目寸光看不远，算不全对面有什么争先手段。

举几盘例子试试：

### 中炮对屏风马互进三兵 ？

1. 炮二平五 马8进7
2. 马二进三 车9平8
3. 车一平二 马2进3
4. 兵三进一 卒3进1

??? note "局面图"
    ![image-20250810191032217](https://raw.githubusercontent.com/DeutschBall/picbed/main/image-20250810191032217.png)

此时红方能否先送七兵然后送三兵，再抬右车巡河抓3卒呢？

不可以，因为黑方可以退炮打车打底相

??? note "局面图"
    ![image-20250810191330146](https://raw.githubusercontent.com/DeutschBall/picbed/main/image-20250810191330146.png)

或者黑方右马盘河也能给卒生根，这一个卒就可以压制红方整个右翼，就像沾了屎的扫把一样。

??? note "局面图"
    ![image-20250810191449101](https://raw.githubusercontent.com/DeutschBall/picbed/main/image-20250810191449101.png)

那么当前局面和五七炮弃双兵的区别在哪？

??? note "对比图"
    ![左五七炮弃双兵 右中炮对屏风马互进三兵](https://raw.githubusercontent.com/DeutschBall/picbed/main/%E5%90%88%E5%B9%B6.png)

当前局面红没有边马，黑卒可以下冲，然后顺势上右马盘河可以保护黑卒，因此红车逮不住黑卒，这两个大兵是纯白给。

并且当前局面下红方左车未出，黑方右车对应也没出，那么黑右炮就可以退到马后面防守反击。在五七炮弃双兵局面下，由于红左直车拉住了黑右侧车炮，黑不能退炮藏到马后面打车。

也就是说这两点还是条件5没有达到。
